Dream-defenders
===============

Tools :

GD:

Ouvrir le fichier stats.xlsx
Renseigner/Modifier les champs grow factor
Exporter sous stats.ods
Executer man.bat
Ouvrir la mainScene dans Unity
Dans Unity cliquer sur Window/StatsManager
Renseigner le champ avec le fichier Editor/XML/content.xml
Cliquer sur ok

LevelGenerator:

Ouvrir le fichier.blend
Executer le script
Générer le type de niveau que vous souhaitez 
Cliquer sur exportFbx
Deplacer le fichier dans le dossier Assets/Modèles/Levels
Drag and drop niveau dans la scene
Dans Unity cliquer sur Window/LevelSetupç 
Renseigner les champs et cliquer sur ok (Prefabs/LevelSetup)

Elements non présents dans le jeu :
-Reconnexion en partie
-Chat : channels

Features spécialisés :
-Chat
-Boutique in-Game
